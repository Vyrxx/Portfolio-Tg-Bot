- Есть ИИ
- Быстрый и удобный

## 🚀 Использование

Поменяй тексты и токены в файлах
Теперь у вас есть портфолио в телеграм боте!

## ⬇️ Установка

Загрузи файлы и открой.
Удостоверься что есть модули:
telebot и google-genai(в requirements)
Что бы скачать их напишите это в терминале:
pip install <package_name>
Поменяй токены в main.py и тексты / изображения helper.py
Запусти main.py
Наслаждайся своим портфолио!
