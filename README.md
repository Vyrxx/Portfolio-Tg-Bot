## 🚀 Usage

Change tokens and texts in main.py file and run
Now you can see your portfolio in Telegram Bot!


## ⬇️ Installation

Download source and open IDE.
In IDE make sure that you have packages such as:
telebot and google(in requirements)
To download them
pip install <package_name>
Change Tokens in main.py and texts or images in helper.py
Run main.py
Enjoy your portfolio!
